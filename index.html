Create Paradise Community Link

Sluit je aan!

Wat is het en hoe werkt het?


Create Paradise community link is een functie om communities op discord te verbinden!
Elk bericht dat jij stuurt wordt naar alle andere verbonden kanalen gestuurd!

Maar, hoe werkt het?
Als de Create Paradise community bot in jouw discord server zit zal elk bericht dat in een verbonden kanaal wordt gestuurd
worden verwijderd, maar dat is niet alles. Je bericht, naam en avatar worden naar onze server gestuurd en via een webhook
naar het hele netwerk worden gestuurd.

Wil jij meer weten over hoe het in elkaar zit?
Alle code is beschikbaar op mijn github!

Maar, hoe word je lid?

Om lid te worden moet je een paar stappen volgen:

  1. Ga naar Mijn github en fork de repository
  2. Vul in bot.js het kanaal-id van het kanaal dat je wil linken in bij "if (message.channel.id === {id} || etcetera)"
  3. Maak een pull request en wacht tot wij dit zien
  4. Als je PR is geaccepteerd stuur je een mail naar koenesmiley@gmail.com met een webhook-url van het kanaal dat je wil linken
  5. Wij gaan voor je aan de slag!
